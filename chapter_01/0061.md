#　写一个函数，将"tom is cat"倒序打印出来，即"cat is tom"

# 参考解答
```java
public class Test61 {
	
	public static void main(String[] args) {
		String str = "tom is cat";
		System.out.println(reverse(str));
	}
	public static String reverse(String str){
		String [] s = str.split(" ");
		String string = "";
		for (int i = s.length-1; i >= 0 ; i--) {
			string+=s[i]+" ";
		}
		return string;
	}

}
```
