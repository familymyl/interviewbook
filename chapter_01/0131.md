# 写出下面程序的执行结果。

```java
class A {
    static {
        System.out.println("1");
    }
    public A() {
        System.out.println("2");
    }
}
class B extends A {
    static {
        System.out.println("a");
    }
    public B() {
        System.out.println("b");
    }
}
public class WorkTest {
    public static void main(String[] args) {
        A ab = new B();
        A ab1 = new B();
    }
}
```

# 参考答案

1
a
2
b
2
b
静态代码块在类加载时被执行一次，只有一次，所以1，a只执行打印了一次
