# 下面代码描述正确的是（ ）

```
public class Divide {
	public static void main(String[] args) {
		System.out.println("17.0/0="+17.0/0);  //1        
	        System.out.println("17/0="+17/0);      //2
	}
}
```
A. 编译出错
B. 编译通过，运行时//1和//2均出现异常
C. 编译通过，运行时//1处得到一个无穷大的值，//2出现异常
D. 编译通过，运行时//1出现异常，//2处得到一个无穷大的值

# 参考答案

C;
