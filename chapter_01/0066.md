# 关于以下程序段，说法正确的是（ ）

```
String s1 = "abc"+"def";                           //1
String s2 = new String(s1);                        //2
if(s1==s2)                                         //3
    System.out.println("==succeeded");             //4
if(s1.equals(s2))                                  //5
    System.out.println(".equals() succeeded");     //6
```
A. 行4与行6都将执行    
B. 行4执行，行6不执行
C. 行6执行，行4不执行  
D. 行4，行6都不执行

# 参考解答

C;
== 比较的是地址
equals 比较的是内容，故s1==s2 为false，s1.equals(s2)为true，行6执行，行4不执行。
